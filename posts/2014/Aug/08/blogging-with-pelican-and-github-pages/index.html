Blogging with Pelican and GitHub pages

Permalink: 2014-08-08 16:59:00 by Kang in Pelican tags: pelican github python

Recording how I created this blog site

Update the deployed github page

# update local file from remote
cd output
git pull origin master

# update site from local
cd
cd blog
pelican -s pelicanconf.py content -o output
cd output
git add .
git commit -m "Update"
git push

custom url using Slug in Markdown, e.g.,

Slug: My-first-painful-pelican-post-in-win-7

custom pelican-themes

cd blog
git clone https://github.com/getpelican/pelican-themes.git
cd pelican-themes  
pelican-themes -i bootstrap2

Create a About page

creat the "pages" fold for individual pages, e.g., a about-me page 

mkdir content/pages # can be done by adding new fold manually

creat a new aboutme.md in the pages fold, and it's done.
